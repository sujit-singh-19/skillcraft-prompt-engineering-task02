# Prompting for Creativity — Task 02

SkillCraft Technology — Prompt Engineering Internship

## Objective
This task explores how prompt structure, tone, and few-shot examples influence
creative AI outputs. Three prompt variants were designed for the same creative
task (generating startup ideas) using different personas and tones, then
compared for tone, creativity, and level of detail.

## Contents
- Task02_Prompting_for_Creativity.docx — Full report including all 3 prompt
  variants, sample outputs, comparison table, and key learnings.

## Variants Compared
- Variant A — Formal VC Advisor persona
- Variant B — Playful Gen-Z Founder persona
- Variant C — Analytical Data-Driven persona

## Key Takeaway
Persona and tone instructions in a prompt shift the style of output more than
the topic itself, while few-shot examples strongly anchor output format and
length.
